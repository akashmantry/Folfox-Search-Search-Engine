<p><b>Ali Illahism</b> (<a href="/wiki/Persian_language" title="Persian language">Persian</a>: <span lang="fa" dir="rtl" xml:lang="fa">علی‌اللّهی</span>‎‎) is a name attributed to a <a href="/wiki/Syncretic" class="mw-redirect" title="Syncretic">syncretic</a> religion which has been practiced in parts of Iranian <a href="/wiki/Luristan" class="mw-redirect" title="Luristan">Luristan</a>, Pakistan, and India which combines elements of <a href="/wiki/Shia_Islam" title="Shia Islam">Shia Islam</a> with older religions. It centers on the belief that there have been successive <a href="/wiki/Incarnations" class="mw-redirect" title="Incarnations">incarnations</a> of the <a href="/wiki/Deity" title="Deity">Deity</a> throughout history, and Ali Ilahees reserve particular reverence for <a href="/wiki/Ali" title="Ali">Ali</a>,<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup> the son-in-law of the Islamic prophet <a href="/wiki/Muhammad" title="Muhammad">Muhammad</a>, who is considered one such incarnation.<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">[2]</a></sup> Various <a href="/wiki/Rites" class="mw-redirect" title="Rites">rites</a> have been attributed as Ali Ilahian, similarly to the <a href="/wiki/Yezidis" class="mw-redirect" title="Yezidis">Yezidis</a>, <a href="/wiki/Alawi" class="mw-redirect" title="Alawi">Ansaris</a>, and all sects whose doctrine is unknown to the surrounding <a href="/wiki/Muslim" title="Muslim">Muslim</a> and <a href="/wiki/Christian" title="Christian">Christian</a> population. Observers have described it as an <a href="https://en.wiktionary.org/wiki/agglomeration" class="extiw" title="wikt:agglomeration">agglomeration</a> of the customs and rites of several earlier religions, including <a href="/wiki/Zoroastrianism" title="Zoroastrianism">Zoroastrianism</a>, historically because travelogues were "evident that there is no definite code which can be described as Ali Illahism"<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">[3]</a></sup></p>
<p>Sometimes Ali-Illahism is used as a general term for the <a href="/wiki/Ghulat" title="Ghulat">several denominations that veneer or deify Ali</a>, like the <a href="/wiki/Alawis" class="mw-redirect" title="Alawis">Alawis</a>, the <a href="/wiki/Alevis" class="mw-redirect" title="Alevis">Alevis</a> or the <a href="/wiki/Yarsanism" title="Yarsanism">Ahl-e Haqq/Yarsanis</a>,<sup id="cite_ref-4" class="reference"><a href="#cite_note-4">[4]</a></sup> others to mean the Ahl-e Haqq.<sup id="cite_ref-5" class="reference"><a href="#cite_note-5">[5]</a></sup></p>
<h2><span class="mw-headline" id="In_the_Dabestan-e_Mazaheb">In the <i>Dabestan-e Mazaheb</i></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ali-Illahism&amp;action=edit&amp;section=1" title="Edit section: In the Dabestan-e Mazaheb">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The <i><a href="/wiki/Dabestan-e_Mazaheb" title="Dabestan-e Mazaheb">Dabestan-e Mazaheb</a></i> (a persian book of the 17th century about the <a href="/wiki/South_Asian" class="mw-redirect" title="South Asian">South Asian</a> religions) presents the Ali Illahians as a sect that respected <a href="/wiki/Muhammad" title="Muhammad">Muhammad</a> and <a href="/wiki/Ali" title="Ali">Ali</a> and discarded the <a href="/wiki/Quran" title="Quran">Quran</a> as it was compiled under <a href="/wiki/Umar" title="Umar">Umar</a>. They avoided killing any animals and believed that the rules allowing the killing of some animals are created by <a href="/wiki/Abu_Bakr" title="Abu Bakr">Abu Bakr</a>, Umar, <a href="/wiki/Uthman_ibn_Affan" class="mw-redirect" title="Uthman ibn Affan">Uthman ibn Affan</a> and their followers.<sup id="cite_ref-6" class="reference"><a href="#cite_note-6">[6]</a></sup></p>
